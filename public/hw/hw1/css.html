Cascading Style Sheets (CSS)

CSS is a style sheet language that is used to help make the HTML presentable. It is designed to help change the layout, colors, and fonts to help improve content accessibility, as well as provide more formatting and control in the presentation of chracterstics.

Syntax

A style sheet consists of a list of rules, which consists of one or more selectors and a declaration block.

Declaration Block

A declaration bloc consists of a list of declarations in braces. Each declaration itself consists of a property, a colon (:), and a value. Each declaration must end with a semicolon (;).

Before CSS

Before CSS, almost all presentational attributes in HTML documents were contained within HTML. All font colors, background styles, element alignments, borders, and sizes, are all explicity described. They had to be repeated for elements with similar styles. With CSS, authors are able to type those styles in a different file, which is much simpler than keeping it intertwined in the HTML.

Before CSS, document authors who wanted to assign typographic characteristics to, for example, all h1 headings had to repeat the HTML presentational markup for each of the occurances of h1. Now, CSS allows authors to be able to select the h1 element and write one declaration block for all the h1elements.
