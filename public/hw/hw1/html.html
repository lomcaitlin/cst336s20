Hyper Text Mockup Language (HTML)

Overview

HTML is a markup language that web browsers use to interpret and compos text, images and other material into visual or audible web pages. There are default characterestics for every HTML element, which can be altered using CSS. HTML is like the bone structure for a website -- it provides the foundation for where different elements are.

HTML helps create structured documents by denoting structural semantics for text such as headings, pragraphs, lists, links, quotes, etc. The elements are portrayed by tags, written inside angle brackets. HTML can also embed programs written in scripting languages like JavaScript. Inclusion of CSS helps to define the look and feel of the web page.

Markup

HTML markup consists of several key components, including those called tags and their attributes. Most HTML tags come in opening and closing pairs, such as <h1> and </h1>

Another important component is the document type declaration, which makes sure that the browser can properly read it as HTML.

Attributes

Most attributes of an element are name-value pairs, separated by =. There are several common attributes that may appear.

The id attribute provides a dociument-wide unique identifyer for an element. This way, the stylesheet can alter it's properties.

The class attribute provides a way of classifying similar elements. It can be used for semantic or presentation purposes.

The style attribute can be used to assign presentational properties to specific particular elements. It's considered better practice to use an element's id or class attributes to change the presentational properities through a CSS file.
