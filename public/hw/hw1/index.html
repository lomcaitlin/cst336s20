What is Front-End Web Development?

Front-end web development is the practice of creating a graphical interface for data through the use of HTML, CSS, and JavaScript, so that users can view and interact with it.


Tools Used

There are are multiple different tools and platforms that can be used to develop the front end of a website.

Hyper Text Markup Language (HTML)

HTML is the backbone for all webpages. It provides the overall framework for how the website will look.

Cascading Style Sheets (CSS)

CSS is what makes the website look visually appealing. It maintains style sheets that sit on top of other style rules.

JavaScript

JavaScript is an event-based programming language that transforms a static HTML page into a dynamic interface. It manipulates a web page to respond to events such as user input.
