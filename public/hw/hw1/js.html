JavaScript

JavaScript, often abbreviated as JS, is a high-level milti-paradigm. JavaScript enables interactive web pages. It supports event-driven, functional, and imperative programming styles. It has APIs for working with text, arrays, dates, regular expressions, and the Document Object Model (DOM).

Client-Side Usage

95% of websites use client-side JavaScript. Scripts are embedded in or included in HTML documents and interact with the DOM. Some examples are loading new page content wihtout reloading the page,animation of page events, interactive content like games and videos, validating input values, and transmitting information about the user's behavior for analytics.

Features

JavaScript supports much of the structured programming syntax from C, like if statements, while loops, switch statements, do while loops. JavaScript is also weakly typed, which means certain types are implicitly cast depending on the operation used. It is also very dynamic.
